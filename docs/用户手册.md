① 注册 → ② 上传简历 → ③ 生成报告

| 浏览器 | 设备 | 网络 |
|--------|------|------|
| Chrome / Edge / Firefox | 电脑（推荐） | 稳定联网 |

## 📋 能力画像

**它做什么**：把你的简历拆成系统能理解的画像。

**怎么用**：

1. 进入 `画像` 页 → 点 **上传简历**
2. 三种方式任选：
   - 📎 上传 PDF（要求文字版，不要扫描图）
   - 🖼️ 上传图片（手机截图可）
   - ✍️ 直接粘贴文字
3. 等 30-60 秒让 AI 解析
4. 完成 **SJT 情景测评**（10 道选择题，5 分钟，评估软技能）

> 📸 `[上传界面截图]`

**💡 小贴士**：SJT 选"真实的你会怎么做"，别选"看起来最好的"——虚高分数会让后续推荐失准。

## 🗺️ 岗位图谱

**它做什么**：45 个岗位的可视化地图，告诉你每个岗位要什么、能往哪走。

**怎么用**：

- **找"我在哪"**：上传简历后，图谱自动标出你的坐标
- **看岗位详情**：点任意节点 → 看核心技能、5 级晋升路径、典型项目、薪资
- **比较两个岗位**：右上角"比较" → 选两个节点
- **设定目标**：点岗位 → "**设为我的目标**" → 后续诊断和报告都以它为基准

> 📸 `[图谱探索截图]`

**💡 小贴士**：别只盯着"热门"岗位——点一下换岗路径，你会发现自己能做的比想的多。

## 🎯 JD 诊断

**它做什么**：看你够不够格投某个招聘岗位。

**怎么用**：

1. 复制招聘网站的 JD 全文
2. 粘贴到右侧 **AI 教练面板** → 系统自动识别 JD 触发诊断
3. 等 20-30 秒看结果

**结果怎么看**：

| 项目 | 含义 |
|------|------|
| **综合分** | 0-100，< 50 先别投，50-70 补齐缺口再投，> 70 可以冲 |
| **四维分** | 技能 50% · 潜力 25% · 基础 15% · 素养 10% |
| **Top 缺口** | 最该补的几项技能 |
| **补法类型** | 📚 learn · 🛠️ practice · 🔀 both |

> 📸 `[JD 诊断结果截图]`

**💡 小贴士**：看缺口旁边的 📚🛠️🔀 图标——**不同缺口用不同方法补**，别一股脑全刷题。

## 📊 发展报告

**它做什么**：生成一份你的个人职业发展蓝图。

**怎么用**：

1. 画像做完后 → 首页点 **生成发展报告**
2. 等 30-60 秒
3. 报告自动包含：AI 评价 · 方向对齐 · 三段式补法地图 · 3 阶段行动计划

**能做什么**：

- ✏️ **编辑**——不满意某段直接改
- 🪄 **AI 润色**——某段写得生硬，点这个重写
- 📥 **导出 PDF**——右上角按钮，一键下载

**多久生成一次**：建议每周 1 次。每份新报告会和上一份对比，让你看到进步。

> 📸 `[发展报告截图]`

**💡 小贴士**：每次报告顶部的 **Delta 对比**（🟢 进步 / 🟡 待提升 / 🔵 下一步）是最有价值的部分，别错过。

## 📝 成长档案

**它做什么**：持续记录你的项目、投递、面试，让规划不停留在"纸面"。

三个 Tab：

### 项目追踪
记录你做过/正在做的项目。做完一个阶段就更新一次。

### 求职追踪
记录每次投递：公司 / 岗位 / 状态（投递 → 笔试 → 面试 → offer）。每轮面试后点 **AI 复盘** 记录反思。

### 档案精修 ⭐
系统识别你简历里**空洞的项目描述**，你来补写。

**空洞的 4 种特征：**
- 🔴 没数字
- 🔴 没成果
- 🟡 太短（< 30 字）
- 🟡 只说"参与"没说"负责"

**补写规则**：必须包含 **数字 + 长度 ≥ 30 字 + 动作动词**（负责/主导/实现），否则提交按钮不亮。

❌ 原文：`参与 XX 后台开发`
✅ 改后：`独立实现 XX 后台 20 个页面，使用 React + TS，通过组件抽离将重复代码减少 40%`

> 📸 `[档案精修截图]`

**💡 小贴士**：**AI 不会帮你代写**——这是故意的。AI 写的简历面试一问就露馅，自己填才能讲得出。

## 🤖 AI 教练

**它做什么**：右侧常驻的对话面板，有问题随时问。

**可以问什么**：

- "我不知道该选前端还是后端"
- "下周有个面试，怎么准备"
- "要不要考研"
- "给我做一份两周学习计划"
- 粘贴 JD 全文 → 自动触发诊断

**3 个用好的技巧**：

1. **说清上下文**："我大三，学了 Java，对前端也感兴趣……" 比光说"给建议"好
2. **粘贴 JD 做诊断**：直接把招聘文案贴进来，系统自动识别
3. **语音输入**：点麦克风图标可以直接说话

## ❓ 常见问题

**Q：简历解析一直转圈圈？**
等 30-60 秒属于正常。超过 2 分钟刷新页面看看；PDF 是扫描版要改用文字粘贴。

**Q：JD 诊断分数很低，是不是没救了？**
大三 50 分很正常。看 **Top 缺口** 那几项，用对应的补法类型（📚/🛠️/🔀）有针对性地补，2-3 个月能提上去。

**Q：档案精修为什么不让 AI 帮我写？**
AI 写的简历你自己讲不出细节，面试一问就穿帮。自己填一次，面试就有料。

**Q：我改了简历但画像没更新？**
画像不会自动同步，需要**重新上传**新简历。

**Q：我的数据会被上传到云端吗？**
画像和报告存在你这台机器的本地数据库。但调用大模型时必要的上下文会发给阿里云百炼（企业级 API 不用于训练模型）。**别在简历里写身份证号**。
