## 各层职责边界

| 层 | 负责 | 不做 |
|----|------|------|
| Router | 文件校验 → 调 service | OCR 判断、ResumeSDK、prompt |
| Service | UploadFile → ResumeFile → 委托 pipeline | 业务逻辑 |
| Pipeline | 编排 5 步流程 | 自己解析、持久化 |
| Extractor | 文件 → 文本 | 生成画像字段 |
| Evidence | 调第三方 API → 证据 dict | 失败不中断主流程 |
| LLMParser | 文本 → ProfileData | 职业判断 |
| Postprocess | 去重去空 + 日期修复 | 技能别名、公司 tier |
